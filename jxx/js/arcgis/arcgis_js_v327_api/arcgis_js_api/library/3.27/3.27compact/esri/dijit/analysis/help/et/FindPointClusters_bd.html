Otsi punktklastreid

Töövahend Otsi punktklastreid leiab ümbritseva müra hulgast punktobjektide klastrid nende ruumilise jaotuse põhjal.

Valitsusväline organisatsioon uurib konkreetset kahjuritest põhjustatud haigust. Selles kasutatakse punktandmestikku, mis esindab uuringupiirkonna kodusid, millest mõned on nakatunud ja mõned mitte. Töövahendi Otsi punktklastreid abil saab analüütik leida nakatunud kodude klastrid, et aidata määrata piirkonda, kus tuleb alustada ravi ja hävitada kahjurid.

Valige kiht klastrite otsimiseks

Punktkiht, milles leiduvad klastrid. Kihid peavad olema tasapinnalises ruumilise lähiseose süsteemis või tasapinnaliseks ruumilise lähiseose süsteemiks peab olema funktsiooni Analüüsikeskkonnad abil määratud tasapinnaline koordinaatsüsteem.

Lisaks kihi valimisele kaardil saate valida ripploendi alumises servas Sirvi kihte, et otsida oma sisust suurandmete failiketta andmestikku või objektikihti.
