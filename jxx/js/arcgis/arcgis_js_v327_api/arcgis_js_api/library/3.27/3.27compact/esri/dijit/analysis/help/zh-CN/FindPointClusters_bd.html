查找点聚类

查找点聚类工具可基于点要素的空间分布查找周围噪点内的点要素聚类。

某非政府组织正在研究一种特别的害虫传播疾病。该组织拥有表示研究区域内家庭的点数据集，其中一些家庭已被感染，而其中一些则没有。通过使用 查找点聚类工具，分析人员可确定受感染家庭的聚类，帮助确定一个区域以开始处理和消灭害虫。

选择要查找聚类的图层

聚类所在的点图层。图层必须位于投影空间参考中，否则必须使用 分析环境将处理空间参考设置为投影坐标系。

除了从地图中选择图层外，还可以在下拉列表的底部选择 浏览图层，以浏览至大数据文件共享数据集或要素图层中的内容。

结果图层名称

将在 我的内容中创建并添加到地图中的图层的名称。默认名称基于工具名称以及输入图层名称。如果该图层已存在，将无法使用该工具。
