Hitta punktkluster

Verktyget Hitta punktkluster hittar kluster av punktgeoobjekt i bakgrundsbrus baserat på deras rumsliga fördelning.

En icke-statlig organisation studerar en viss insektsburen sjukdom. Den har ett punktdataset som representerar hushållen i ett undersökningsområde, där vissa är angripna och andra inte. Med verktyget Hitta punktkluster kan en analytiker fastställa kluster med smittade hushåll och hjälpa dig att lokalisera ett område där behandlingen och utrotning av skadedjuren ska inledas.

Välj det lager som klustren ska hittas från

Det punktlager som klustren kommer att hittas i. Lagren måste finnas i en projicerad spatial referens eller så måste den spatiala referensen vara inställd på ett projicerat koordinatsystem med Analysmiljö.

Förutom att välja ett lager från kartan kan du välja Bläddra bland lager längst ned i listrutan för att bläddra till ditt innehåll efter ett big data-fildelningsdataset eller geoobjektslager.
