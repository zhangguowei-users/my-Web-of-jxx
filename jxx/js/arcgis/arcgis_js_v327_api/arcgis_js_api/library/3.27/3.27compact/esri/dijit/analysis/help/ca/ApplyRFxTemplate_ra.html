Aplica la plantilla de funció de ràster

Aquesta eina processa les imatges amb la funció o la cadena de funcions, tal com s'especifica a la plantilla de funció de ràster.

Entrades

Les entrades us permeten seleccionar les capes que voleu processar, així com configurar tots els paràmetres associats a les funcions de ràster de la cadena de processament.

Nom de la capa de resultats
