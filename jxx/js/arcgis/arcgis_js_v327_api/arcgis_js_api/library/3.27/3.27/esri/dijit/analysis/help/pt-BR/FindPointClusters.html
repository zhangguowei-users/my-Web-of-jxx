Localizar Agrupamentos de Pontos

Esta ferramenta encontra conjuntos de feições de ponto dentro do ruído circundante com base em sua distribuição espacial.

Se Utilizar uma extensão de mapa atual estiver selecionada, somente as feições na camada de ponto de entrada dentro da extensão de mapa atual serão analisadas. Se não selecionada, todas as feições na camada de ponto de entrada serão analisadas, mesmo se estiverem fora da extensão de mapa atual.

Escolher a camada na qual os agrupamentos serão calculados

A camada de ponto onde os agrupamentos serão encontrados.

Número mínimo de pontos a ser considerado como um agrupamento

O número mínimo de feições a ser considerado no agrupamento. Qualquer agrupamento com menos feições que o fornecido será considerado ruído.

Limitar o intervalo de pesquisa para (opcional)

A distância máxima para considerar feições para fazerem parte do mesmo agrupamento. Se deixado em branco, a distância padrão utilizada será a maior distância do núcleo encontrada no conjunto de dados, excluindo estas distâncias principais no 1% superior (excluindo as distâncias principais mais extremas).

Nome da camada resultante

Este é o nome da camada que será criada em Meu Conteúdo e adicionada no mapa. O nome padrão é baseado no nome da ferramenta e o nome da camada de entrada. Se a camada já existir, você será solicitado para fornecer outro nome.

Ao utilizar a caixa suspensa Salvar resultado em, é possível especificar o nome de uma pasta em Meu Conteúdo onde o resultado será salvo.
