Vektoros elem átalakítása raszterré

This tool converts features to a raster dataset.

Any feature class containing point, line, or polygon features can be converted to a raster dataset.

A kimeneti raszteres pixel értékének meghatározásához a cella középpontját használja.

A kimeneti raszter típusát a bemeneti mező típusa határozza meg. Ha a mező integer, a kimeneti raszter is integer lesz; ha lebegőpontos érték, akkor a kimenet is lebegőpontos érték lesz.

Válassza ki az átalakítani kívánt vektoros réteget

A raszteres adathalmazzá átalakítani kívánt bemeneti vektoros réteg.

Válasszon mezőt

Válassza ki a mezőt, amelyet az értékek kimeneti raszterhez rendeléséhez szeretne használni.

Kimeneti cellaméret

Enter the cell size and unit for the output rasters.

A rendelkezésre álló mértékegységek: mérföld, láb, kilométer és méter. Az alapértelmezett mértékegyég a láb.

Eredmény réteg neve

The name of the layer that will be created in My Content and added to the map. Az alapértelmezett név az eszköz nevétől és a bemenő réteg nevétől függ. Ha a réteg már létezik, akkor a program megkéri, hogy adjon meg egy másik nevet.

Az Eredmény mentési helye legördülő mezőben megadhatja a Saját tartalom mappájának nevét, ahová az eszköz menti az eredményt.
