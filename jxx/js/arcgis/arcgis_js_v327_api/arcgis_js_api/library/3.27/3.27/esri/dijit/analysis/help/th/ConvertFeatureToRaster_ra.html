แปลงเวกเตอร์เป็นแรสเตอร์

เครื่องมือนี้แปลงเวกเตอร์เป็นแรสเตอร์ดาต้าเซ็ต

ฟีเจอร์คลาสประกอบไปด้วยข้อมูล จุด, เส้น, หรือพื้นที่รูปปิด ฟีเจอร์เหล่านี้สามารถแปลงเป็นแรสเตอร์ดาต้าเซ็ตได้

เซลล์ตรงกลางจะใช้เป็นตัวกำหนดค่าของพิกเซลแรสเตอร์เอาต์พุต

ประเภทของฟิลด์ที่ใส่จะเป็นตัวกำหนดประเภทของแรสเตอร์เอาต์พุต ถ้าฟิลด์เป็นจำนวนเต็ม, แรสเตอร์เอาต์พุตก็จะเป็นจำนวนเต็ม; ถ้าเป็นจุดทศนิยม, เอาต์พุตก็จะเป็นจุดทศนิยม

เลือกชั้นข้อมูลเวกเตอร์ที่ต้องการแปลง

ใส่ชั้นข้อมูลฟีเจอร์ที่ต้องการแปลงเป็นแรสเตอร์ดาต้าเซ็ต

เลือกฟิลด์

เลือกฟิลด์ที่จะใช้เป็นตัวกำหนดค่า values ของแรสเตอร์เอาต์พุต

ขนาดของเซลล์เอาต์พุต

ป้อนขนาดเซลล์และหน่วยของแรสเตอร์เอาต์พุต

หน่วยที่ใช้ได้คือ ไมล์ ฟุต กิโลเมตรและเมตร ค่าเริ่มต้นคือฟุต

ชื่อของชั้นข้อมูลผลลัพธ์

ชื่อของชั้นข้อมูลที่จะถูกสร้างขึ้นใน เนื้อหาของฉัน และเพิ่มไปยังแผนที่ ชื่อเริ่มต้นจะขึ้นอยู่กับชื่อเครื่องมือและชื่อชั้นข้อมูลอินพุต ถ้าชั้นข้อมูลนี้มีอยู่แล้ว คุณจะถูกถามให้ตั้งชื่อใหม่

ใช้ บันทึกผลลัพธ์ใน รายการตัวเลือก เพื่อระบุชื่อของโฟล์เดอร์ใน เนื้อหาของฉัน เมื่อบันทึกผล
