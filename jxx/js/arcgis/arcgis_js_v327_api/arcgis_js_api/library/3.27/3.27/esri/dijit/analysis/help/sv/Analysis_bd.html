GeoAnalytics fördelar analysarbetet över ArcGIS GeoAnalytics-servern. På så sätt kan du analysera större mängder data snabbare genom att utnyttja kraften hos flera datorer. För tillfället finns följande verktyg tillgängliga:

  • Aggregera punkter
  • Skapa multivariabelt rutnät
  • Förena geoobjekt
  • Återskapa spår
  • Summera attribut
  • Summera inom
  • Geokoda platser från tabell
  • Identifiera händelser
  • Hitta liknande platser
  • Beräkna densitet
  • Hitta punktkluster
  • Hitta hotspots
  • Skapa buffertar
  • Bifoga data
  • Beräkna fält
  • Kopiera till datalager
  • Överlagra lager

GeoAnalytics fokuserar på att analysera stora mängder data med betoning på både geografiska och tidsbaserade mönster.

Summera data

Verktygssetet Summera data innehåller verktyg som beräknar totalt antal, längd, yta och enkel beskrivande statistik för geoobjekt och deras attribut inom områden eller nära andra geoobjekt.

  • Aggregera punkter beräknar statistik om punkter som hamnar inom angivna områden eller lagringsplatser.
  • Skapa multivariabelt rutnät genererar ett rutnät av kvadratiska eller sexkantiga klasser och beräknar värden för varje klass baserat på närhet till varje indatalager.
  • Förena geoobjekt beräknar statistik om geoobjekt som delar en rums-, tids- eller attributrelation med andra geoobjekt.
  • Återskapa spår beräknar statistik om punkter eller polygoner som tillhör samma spår och återskapar indata till spår.
  • Summera attribut beräknar statistik om geoobjekts- eller tabelldata som delar attribut.
  • Summera inom beräknar statistik för områdesgeoobjekt och attribut som överlappar varandra.

Aggregera punkter

Det här verktyget använder ett lager med punktgeoobjekt och antingen ett lager med områdesgeoobjekt eller lagringsplatser som definieras av ett angivet avstånd för att avgöra vilka punkter som hamnar i vilket område eller lagringsplats och beräkna statistik om alla punkter i varje område eller lagringsplats. Du kan också använda tidsdelning med verktyget.

Följande är exempel:

  • Med hjälp av punkter där brott har begåtts vill du räkna ut antalet brott per län eller annat administrativt distrikt.
  • Hitta de högsta och lägsta intäkterna för franchisebutiker med 100-kilometersklasser


Skapa multivariabelt rutnät

Verktyget Skapa multivariabelt rutnät genererar ett rutnät av kvadratiska eller sexkantiga klasser och beräknar variabler för varje klass baserat på närhet till ett eller flera indatalager.

Följande är exempel:

  • Givet flera lager med kollektivtrafikinfrastruktur, vilken del av staden är minst tillgänglig med allmänna kommunikationer?
  • Givet flera lager med vattenleder, t.ex. sjöar och floder, vad heter den vattenmassa som befinner sig närmast varje plats i USA?
  • Givet ett lager med hushållens inkomster, var i USA är inkomstvariationerna i de omgivande 50 km som störst?


Förena geoobjekt

Du kan använda antingen geoobjektslager eller tabelldata för att förena geoobjekt och poster baserat på specifika relationer mellan indatalagren eller tabellerna. Föreningar avgörs av rums-, tids- och attributrelationer. Det går också att beräkna summeringsstatistik.

Följande är exempel:

  • Med punktplatser för brottsfall med en tid, kan du förena brottsdata med sig självt genom att ange en geografisk relation för brott inom 1 kilometer från varandra och som inträffade inom 1 timme från varandra för att avgöra om det finns en serie brott nära varandra i rum och tid.
  • Om du får en tabell med postnummer med demografisk information och områdesgeoobjekt som representerar bostadshus, kan du förena den demografiska informationen med bostäderna så att varje bostad nu har informationen.


Återskapa spår

Med ett tidsaktiverat lager med punkt- eller polygongeoobjekt som representerar en tidpunkt, avgör verktyget vilka indatageoobjekt som hör hemma i ett spår och ordnar indata i tidsordning. Det går också att beräkna statistik för indatageoobjekten inom varje spår.

Följande är exeampel:

  • Beräkna orkanens medelvindhastighet och maximala vindtryck utifrån punktplatser och tid för orkanmätningar.
  • Använd GPS-mätningar från fartyg för att rekonstruera fartygsrutter som linjer.


Summera attribut

Verktyget summerar statistik för angivna fält med antingen geoobjekts- eller tabelldata.

Följande är exempel:

  • Använd platser för livsmedelsbutiker med fältet COMPANY_NAME för att summera butikerna efter företagsnamn för att fastställa statistik för varje företag.
  • Använd en tabell med livsmedelsbutiker med fälten COMPANY_NAME och COUNTY för att summera butikerna efter företagsnamn och län för att fastställa statistik för varje företag inom varje län.


Summera inom

Hittar områden (och delar av områden) som överlappar mellan två lager och beräknar statistik om överlappningen.

Följande är exempel:

  • Med ett lager för avrinningsområden och ett lager med markanvändningsområden enligt markanvändningstyp kan du beräkna den totala arealen för de olika markanvändningstyperna för varje avrinningsområde.
  • Med ett lager för fastigheter i ett län och ett lager kommungränser kan du sammanfatta snittvärdet för de tomma fastigheterna i varje kommun.


Hitta platser

Dessa verktyg används för att identifiera områden som uppfyller ett antal olika kriterier du anger.

  • Identifiera händelser fungerar med tidsaktiverade lager för att avgöra vilka geoobjekt som uppfyller ett visst villkor.
  • Geokoda platser från tabell konverterar adresser till koordinater.
  • Hitta liknande platser hittar de platser som är mest lika en eller flera referensplatser baserat på kriterier som du anger.

Identifiera händelser

Det här verktyget fungerar med ett tidsaktiverat lager med punkter, linjer, områden eller tabeller som representerar ett ögonblick i tiden. Genom att använda geoobjekt som ordnats i tidsordning, så kallade spår, avgör det här verktyget vilka geoobjekt som utgör händelser som är intressanta att hitta. Villkoren som du anger avgör vad som betraktas som händelser.

Följande är exempel:

  • Identifiera orkanmätningar som är Kategori 4-orkaner baserat på ett attribut för vindhastighet.
  • Genom att använda GPS-mätningar för fartygsrörelser med ett fält som representerar fartygets hastighet, kan du hitta mätningar som är 1,5 gånger större än medelvärdet för de fem tidigare mätningarna.


Geokoda platser från tabell

Konverterar adresser till koordinater. Använd det här verktyget för tabeller i big data-fildelningar.


Hitta liknande platser

Hitta liknande platser genom att mäta likheten mellan platser i kandidatsöklagret och en eller flera referensplatser utifrån kriterier som du anger.

Följande är exempel:

  • Hitta de tio mest liknande butikerna genom att undersöka antalet anställda och årsförsäljningen.
  • Hitta de 100 mest liknande städerna genom att undersöka relationen mellan befolkning, årstillväxt och skatteintäkter.


Analysera mönster

Dessa verktyg hjälper dig identifiera, mäta och visualisera geografiska mönster i dina data.

  • Beräkna densitet tar kända antal av ett fenomen och sprider ut dessa antal över kartan.
  • Med Hitta hotspots kan du identifiera statistiskt signifikant klustring i de geografiska mönstren för dina data.
  • Hitta punktkluster hittar kluster av punktgeoobjekt i bakgrundsbrus baserat på deras rumsliga fördelning.

Beräkna densitet

Med verktyget Beräkna densitet kan du skapa en densitetskarta av punktgeoobjekt genom att sprida ut kända antal av något fenomen (som representeras som attribut för punkterna) över kartan. Resultatet är ett lager med områden som representerar densiteten.

Följande är exempel:

  • Beräkna densitet för sjukhus inom en kommun. Resultatlagret visar områden med hög och låg tillgång till sjukhus, och den här informationen kan användas för att avgöra var nya sjukhus bör byggas.
  • Identifiera områden där risken för skogsbränder är stor baserat på platsen för tidigare skogsbränder.
  • Hitta orter som ligger långt från större motorvägar för att kunna planera var nya vägar bör byggas.


Hitta punktkluster

Verktyget Hitta punktkluster hittar kluster av punktgeoobjekt i bakgrundsbrus baserat på deras rumsliga fördelning.

Följande är exempel:

  • Hitta kluster av skadedjursangripna hushåll för att hjälpa till med målinriktade utrotningskampanjer.
  • Informera och agera på räddnings- och evakueringsbehov baserat på storlek och placering av kluster med geolokaliserade tweets efter naturkatastrofer eller terrorattacker.


Hitta hotspots

Med verktyget Hitta hotspots kan du fastställa om det finns statistiskt signifikanta kluster i det geografiska mönstret för dina data.

  • Är dina punkter (brottsfall, träd, trafikolyckor) verkligen klustrade? Hur kan du vara säker på det?
  • Har du verkligen upptäckt en statistiskt signifikant hotspot (för utgifter, barndödlighet, ständigt höga provresultat), eller skulle kartan berätta något annat om du ändrade hur den symboliseras?
Verktyget Hitta hotspots hjälper dig besvara de här frågorna med säkerhet.


Använd närhet

De här verktygen hjälper dig besvara en av de vanligaste frågorna vid geografiska analyser: Vad ligger nära vad?

  • Skapa buffertar skapar områden med ett angivet avstånd från geoobjekt.

Skapa buffertar

En buffert är ett område som täcker ett givet avstånd från ett punkt-, linje- eller polygongeoobjekt.

Följande är exempel:

  • Använd linjära flodgeoobjekt för att buffra varje flod med 50 gånger flodens bredd för att fastställa en föreslagen strandgräns.
  • Använd områden som representerar länder för att buffra varje land med 200 nautiska mil för att fastställa sjögränsen.


Hantera data

Dessa verktyg används för den dagliga hanteringen av geografiska data och data i tabellform.

  • Bifoga data lägger till geoobjekt i ett befintligt driftat geoobjektlager.
  • Beräkna fält skapar ett nytt fält eller modifierar ett befintligt fält och skapar ett nytt lager i dina innehåll.
  • Kopiera till datalager kopierar data till ditt ArcGIS Data Store och skapar ett lager i dina innehåll.
  • Överlagra lager kombinerar två eller flera lager till ett enda lager.

Kopiera till datalager

Kopierar ett indatageoobjektlager eller en tabell till ett ArcGIS Data Store och skapar ett lager i dina innehåll i ArcGIS Enterprise.

Följande är exempel:

  • Kopiera en samling .csv filer i en big data-fildelning till det rumstemporala datalagret för visualisering.
  • Kopiera geoobjekten i den aktuella kartutbredningen som finns lagrade i det rumstemporala datalagret till det relationella datalagret.


Beräkna fält

Beräknar värden för ett nytt eller befintligt fält och skapar ett nytt lager i dina innehåll i ArcGIS Enterprise.

Följande är exempel:

  • Ändra ett befintligt fält med namnet total så att det är summan av intäkter från fälten total_2016, total_2017 och total_2018.
  • Skapa ett nytt fält för att kategorisera risknivåer baserat på fältvärden som windspeed och pollutant.


Överlagra lager

Överlagring kombinerar två eller flera lager till ett enda lager. Överlagring används för att besvara en av de vanligaste frågorna inom geografin: Vad finns ovanpå vad? Följande är exempel:

  • Vilka fastigheter finns inom gränserna för den 100-åriga flodslätten? (Vilket är ett annat sätt att säga ovanpå.)
  • Vilken typ av markanvändning finns på vilken jordtyp?
  • Vilka brunnar finns på övergivna militärbaser?


Bifoga data

Lägger till geoobjekt i ett befintligt driftat lager i ditt innehåll i ArcGIS Enterprise.

Följande är exempel:

  • Uppdatera geoobjektlagret 2018_Sales varje månad när dina rapporter blir tillgängliga.
  • Lägg till dataset för orkanspårning från år 2016 och 2017 i en huvuddataset för orkaner.
  • Samla in och lägg till dataset från olika källor med olika scheman med hjälp av anpassad fältkartering.
