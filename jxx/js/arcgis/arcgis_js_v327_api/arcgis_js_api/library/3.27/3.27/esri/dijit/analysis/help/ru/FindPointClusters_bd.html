Поиск кластеров точек

Инструмент Поиск кластеров точек определяет среди точечных объектов кластеры в окружающем шуме на основе их пространственного распределения.

Негосударственная компания изучает частный случай распространения заболевания. Есть точечный набор данных, представляющий домохозяйства на изучаемой территории, некоторые из которых заражены, а некоторые - нет. С помощью инструмента Поиск кластеров точек специалист может определить кластеры зараженных домохозяйства, это поможет выявить области, где надо начинать лечение и борьбу с переносчиками.

Выберите слой, для которого будут найдены кластеры

Точечный слой, в котором нужно выявить кластеры. У слоев должна быть задана пространственная привязка или пространственная привязка обработки должна быть задана как система координат проекции с помощью Параметров среды анализа.

Кроме выбора слоя из вашей карты, можно щелкнуть Выбрать слои в нижней части ниспадающего списка, чтобы найти свои ресурсы для набора данных или слоя объектов файлового хранилища больших данных.

Минимальное число точек, которые будут считаться кластером

Минимальное число объектов для включения в кластер.

Ограничить диапазон поиска до

Максимальное расстояние между объектами, которые могут быть включены в один кластер.

Имя слоя результата

Имя слоя, который будет добавлен в Мои ресурсы и на карту. Имя слоя по умолчанию зависит от имени инструмента и имени входного слоя. Если слой уже существует, произойдет сбой.

С помощью ниспадающего списка поля Сохранить результат в вы можете задать имя папки в разделе Мои ресурсы, в которую будет записан результат.
