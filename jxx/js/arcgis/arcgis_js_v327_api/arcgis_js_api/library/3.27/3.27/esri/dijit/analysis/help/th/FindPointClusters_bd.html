Find Point Clusters

เครื่องมือ Find Point Clusters จะค้นหาฟีเจอร์ของจุดภายในเสียงแวดล้อมโดยอิงอยู่กับการกระจายในเชิงพื้นที่

องค์กรนอกภาครัฐกำลังศึกษาถึงโรคที่มีแมลงเป็นพาหะเป็นกรณีพิเศษ โดยองค์กรมีชุดข้อมูลจุุดที่แสดงถึงครัวเรือนในพื้นที่ศึกษา ซึ่งบางพื้นที่มีการติดเชื้อ และบางพื้นที่ไม่มีการติดเชื้อ เมื่อใช้เครื่องมือ Find Point Clusters การวิเคราะห์จะสามารถระบุการกระจุกตัวของครัวเรือนที่ติดเชื้อ เพื่อช่วยระบุพื้นที่ที่จะเริ่มการบำบัดและขจัดแมลง

เลือกชั้นข้อมูลที่จะพบกลุ่ม

เราจะพบชั้นข้อมูลจุดที่กระจุกตัว ชั้นข้อมูลต้องอยู่ในการอ้างอิงเชิงพื้นที่ที่คาดการณ์หรือมีชุดการอ้างอิงเชิงพื้นที่ที่ประมวลผลไปยังระบบพิกัดโปรเจคชันโดยใช้ สภาพแวดล้อมการวิเคราะห์

นอกเหนือจากการเลือกชั้นข้อมูลจากแผนที่คุณสามารถเลือก เบราว์เลเยอร์ ที่ด้านล่างของรายการแบบเลื่อนลงเพื่อเรียกดูเนื้อหาของคุณสำหรับชุดข้อมูลชุดข้อมูลข้อมูลขนาดใหญ่หรือฟีเจอร์เลเยอร์

จำนวนจุดต่ำสุดที่จะถือว่าเป็นกลุ่ม

จำนวนต่ำสุดของฟีเจอร์ไปยังจุดกำเนิดของกลุ่ม

จำกัดช่วงในการค้นหาที่

ระยะทางสูงสุดที่จะพิจารณาฟีเจอร์ที่จะเป็นส่วนหนึ่งของกลุ่มเดียวกัน

ชื่อของชั้นข้อมูลผลลัพธ์

ชื่อ ของชั้นข้อมูลที่จะถูกสร้างขึ้นใน เนื้อหาของฉัน และเพิ่มไปยังแผนที่ ชื่อเริ่มต้นจะขึ้นอยู่กับชื่อเครื่องมือและชื่อชั้นข้อมูลอินพุต ถ้าชั้นข้อมูลมีอยู่แล้วเครื่องมือจะล้มเหลว

ใช้ บันทึกผลลัพธ์ใน รายการตัวเลือก เพื่อระบุชื่อของโฟล์เดอร์ใน เนื้อหาของฉัน เมื่อบันทึกผล
