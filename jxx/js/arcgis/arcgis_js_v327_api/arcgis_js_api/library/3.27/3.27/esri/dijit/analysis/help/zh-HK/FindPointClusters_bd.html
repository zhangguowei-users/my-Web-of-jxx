尋找點叢集

尋找點叢集工具可在基於其空間分佈的周圍噪訊內，尋找點圖徵的叢集。

一個非政府組織正在研究一種特殊的害蟲傳播疾病。它有一個代表研究區域中家庭的點資料集，其中一些已經被感染，另一些則沒有。藉由使用 尋找點叢集工具，分析人員可判定受感染的家庭叢集，以協助鎖定某個區域來開始治療和消滅害蟲。

選擇要尋找的叢集所在的圖層

將在其中尋找叢集的點圖層。圖層必須位於投影空間參考中，或必須使用 分析環境，將處理空間參考設為投影座標系統。

除了從地圖選擇圖層外，您也可以選擇下拉式清單底部的 瀏覽圖層，以瀏覽到大數據檔案分享資料集或圖徵圖層的內容。
