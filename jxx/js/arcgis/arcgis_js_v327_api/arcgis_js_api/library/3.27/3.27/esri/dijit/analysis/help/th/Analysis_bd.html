GeoAnalytics กระจายงานวิเคราะห์ลงบน ArcGIS GeoAnalytics Server ของคุณ ซึ่งช่วยให้คุณสามารถวิเคราะห์ข้อมูลได้มากขึ้น รวดเร็วขึ้น ผ่านการทำงานจากเครื่องหลายเครื่อง เครื่องมือต่อไปนี้ สามารถใช้ได้ในปัจจุบัน:

  • รวมจุด
  • สร้างตารางหลายตัวแปร
  • รวมฟีเจอร์
  • สร้างแทร็ก
  • สรุปค่าตารางข้อมูลเชิงบรรยาย
  • สรุปข้อมูลภายในพื้นที่
  • ตำแหน่งจีโอโค้ดจากตาราง
  • ตรวจหาเหตุการณ์
  • ค้นหาตำแหน่งที่ใกล้เคียง
  • คำนวณความหนาแน่น
  • Find Point Clusters
  • หาจุดความร้อน
  • สร้างบัฟเฟอร์
  • ข้อมูลภาคผนวก (Append Data)
  • คำนวณฟิลด์
  • คัดลออกไปยังข้อมูล
  • ซ้อนทับชั้นข้อมูล

GeoAnalytics เน้นที่การวิเคราะห์ข้อมูลขนาดใหญ่ด้วยการให้ความสำคัญที่รูปแบบเชิงพื้นที่และเชิงเวลา

สรุปข้อมูล

ชุดเครื่องมือในการสรุป ประกอบไปด้วยเครื่องมือต่างๆในการคำนวณจำนวณนับทั้งหมด, ความยาว, พื้นที่ และสถิติเชิงพรรณนาพื้นฐานของฟีเจอร์ และข้อมูลเชิงบรรยาย ในพื้นที่หรือใกล้กับฟีเจอร์อื่น

  • สรุปจุด คำนวณค่าสถิติเกี่ยวกับจุด หรือตกอยู่ในพื้นที่ที่ระบุหรือถัง
  • สร้างตารางหลายตัวแปร สร้างตารางของถังรูปสี่เหลี่ยมจัตุรัสหรือหกเหลี่ยม และคำนวณค่าสำหรับแต่ละถังตามความใกล้ชิดของชั้นข้อมูลอินพุตแต่ละชั้น
  • รวมฟีเจอร์ คำนวณค่าสถิติเกี่ยวกับฟีเจอร์ที่แบ่งปันข้อมูลเชิงพื้นที่, เวลา หรือความสัมพันธ์ในตารางกับฟีเจอร์อื่น
  • สร้างแทร็ก คำนวณค่าสถิติเกี่ยวกับจุดหรือพื้นที่ ที่ขึ้นอยู่กับแทร็กเดียวกัน และสร้างค่านำเข้าใหม่เข้าสู่แทร็ก
  • สรุปค่าตารางข้อมูลเชิงบรรยาย คำนวณค่าสถิติเกี่ยวกับฟีเจอร์หรือข้อมูลตารางที่แบ่งปันรายละเอียด
  • การสรุปข้อมูลที่อยู่ภายใน คำนวณค่าสถิติของพื้นที่ที่สนใจหรือข้อมูลคุณลักษณะที่มีการซ้อนทับกัน

ผลรวมจุด

ใช้ชั้นข้อมูลจุด และชั้นข้อมูลพื้นที่ หรือสี่เหลี่ยม โดยการกำหนดระยะทาง เครื่องมือนี้จะกำหนดจุดตกอยู่ในแต่ละพื้นที่หรือสี่เหลี่ยม และคำนวณสถิติเกี่ยวกับจุดทั้งหมดที่อยู่ภายในแต่ละพื้นที่ หรือสี่เหลี่ยม คุณอาจเลือกใช้การหั่นเวลากับเครื่องมือนี้

ต่อไปนี้คือ ตัวอย่าง:

  • แสดงจุดเกิดเหตุอาชญากรรม แล้วนับจำนวนอาชญากรรมที่เกิดขึ้นในหนึ่งเขตการปกครองหรือในท้องที่นั้น ๆ
  • หารายได้ต่อเดือนสูงสุดและต่ำสุด สำหรับตำแหน่งของแฟรนไชส์โดยใช้ถังขนาด 100-ตารางกิโลเมตร


สร้างตารางหลายตัวแปร

เครื่องมือสร้างตารางหลายตัวแปรจะสร้างตารางของถังรูปสี่เหลี่ยมจัตุรัสหรือหกเหลี่ยม และคำนวณค่าสำหรับแต่ละถังตามความใกล้ชิดของชั้นข้อมูลอินพุตหนึ่งชั้นหรือหลายชั้น

ต่อไปนี้คือตัวอย่าง:

  • เมื่อดูจากชั้นข้อมูลหลาย ๆ ชั้นของโครงสร้างระบบขนส่งสาธารณะ ส่วนใดของเมืองที่ระบบขนส่งสาธารณะเข้าถึงได้
  • เมื่อดูจากชั้นข้อมูลของทางน้ำ เช่น ทะเลสาบและแม่น้ำ แม่น้ำชื่ออะไรที่อยู่ใกล้แต่จะสถานที่ในสหรัฐฯ มากที่สุด
  • เมื่อดูจากชั้นข้อมูลของรายได้ในครัวเรือน ที่ใดในสหรัฐฯ ที่้้เป็นตัวแปรของรายได้ในพื้นที่รอบ ๆ 50 ไมล์มากที่สุด


รวมข้อมูล

โดยใช้ทั้งชั้นข้อมูล หรือ ข้อมูลตาราง คุณสามารถรวมฟีเจอร์ และบันทึกตามความสัมพันธ์เฉพาะระหว่างชั้นข้อมูลอินพุต หรือตารางได้ การรวมจะถูกกำหนดโดยพื้นที่ ช่วงเวลา และความสัมพันธ์ของแอตทริบิวต์ และจะสรุปสถิติตามที่เลือกคำนวณได้

ต่อไปนี้คือ ตัวอย่าง:

  • มีการให้ตำแหน่งสถานที่จุดของการเกิดเหตุอาชญากรรมที่มีเวลา การรวมข้อมูลอาชญากรรมเองที่มีความสัมพันธ์ภายใน 1 กิโลเมตรจากเหตุการณ์อื่น ๆ ที่เกิดขึ้นภายใน 1 ชั่วโมงของเหตุการณ์อื่น เพื่อตรวจสอบว่ามีลำดับของการก่ออาชญากรรมที่ใกล้เคียงกันหรือไม่ในแต่ละพื้นที่ และเวลา
  • ได้รับตารางรหัสไปรษณีย์ที่มีข้อมูลและพื้นที่ประชากรศาสตร์ และมีข้อมูลพื้นที่ที่แสดงขอบเขตอาคารที่อยู่อาศัย ต้อการรวมข้อมูลประชากร เข้ากับขอบเขตที่อยู่อาศัย เพื่อดูข้อมูลการอยู่อาศัยแต่ละบ้าน


สร้างแทร็กซ้ำ

โดยใช้ชั้นเวลาที่เปิดใช้งานของจุดหรือรูปหลายเหลี่ยมคุณสมบัติที่เป็นตัวแทนของ สิ่งที่เกิดขึ้น ในเวลาที่เครื่องมือนี้กำหนดคุณสมบัติการป้อนข้อมูลที่อยู่ในการติดตามและจะสั่งซื้อปัจจัยการผลิตตามลำดับเวลา สถิติเป็นตัวเลือกในการคำนวณสำหรับข้อมูลฟีเจอร์แต่ละแทร็ค

ต่อไปนี้คือตัวอย่าง:

  • ตำแหน่งสถานที่ที่กำหนด และเวลาของการวัดพายุเฮอริเคนคำนวณตามความเร็วลมเฉลี่ย และแรงดันลมสูงสุดของพายุเฮอริเคน
  • ใช้การวัดจีพีเอสของเรือเพื่อ สร้างทางเดินเรือในรูปของเส้นอีกครั้ง


สรุปแอตทริบิวต์

ใช้ได้ทั้งข้อมูลที่เป็นฟีเจอร์ และตาราง เครื่องมือนี้สรุปสถิติเฉพาะฟิลด์ที่เลือก

ต่อไปนี้คือตัวอย่าง:

  • ตำแหน่งของร้านขายของชำกับฟิลด์ COMPANY_NAME สรุปร้านค้าโดยใช้ชื่อ บริษัท โดยจะตรวจสอบสถิติของแต่ละบริษัท
  • ได้ตารางของร้านขายของชำกับฟิลด์ COMPANY_NAME และ COUNTY สรุปร้านค้าโดยชื่อ บริษัท และเขตเพื่อตรวจสอบสถิติของแต่ละ บริษัท ในแต่ละเขต


สรุปผลรวมภายใน

ค้นหาพื้นที่ (และบางส่วนของพื้นที่) ที่ซ้อนทับกันระหว่างชั้นข้อมูล 2 ชั้น และคำนวณค่าสถิติที่มีการซ้อนทับกัน

ต่อไปนี้คือตัวอย่าง:

  • กำหนดชั้นข้อมูลพื้นที่รับน้ำและชั้นข้อมูลการใช้ที่ดินตามประเภทที่ดิน แล้วคำนวณพื้นที่การใช้ที่ดินทั้งหมดในแต่ละพื้นที่รับน้ำ
  • กำหนดชั้นข้อมูลพื้นที่เมืองและชั้นข้อมูลขอบเขตเมือง แล้วสรุปค่าเฉลี่ยของพื้นที่ว่างเปล่าที่มีในแต่ละเมือง


ค้นหาตำแหน่ง

เครื่องมือนี้ถูกใช้เพื่อระบุพื้นที่ ที่พบจำนวนของเงื่อนไขที่แตกต่างที่คุณระบุ.

  • ตรวจหาเหตุการณ์ ทำงานกับชั้นข้อมูลที่เปิดใช้งานเวลา เพื่อกำหนดว่าฟีเจอร์ใดที่ตรงตามเงื่อนไขที่ระบุ
  • ตำแหน่งจีโอโค้ดจากตาราง แปลงที่อยู่เป็นพิกัด
  • ค้นหาตำแหน่งที่ใกล้เคียง ค้นหาตำแหน่งที่ใกล้เคียงกับอีกตำแหน่งหนึ่งหรือตำแหน่งอื่น ๆ ที่อ้างอิงจากเงื่อนไขที่คุณกำหนดขึ้น

ตรวจหาเหตุการณ์

เครื่องมือนี้ทำงานด้วยจุดชั้นข้อมูลที่ค่าเวลาเปิดใช้งาน เส้น พื้นที่หรือตารางซึ่งแสดงแทนค่าคงที่ในเวลา การใช้ฟีเจอร์ที่เรียงตามลำดับ ซึ่งเรียกกว่าแทร็ก จะทำให้เครื่องมือนี้กำหนดได้ว่าฟีเจอร์ใดเป็นเหตุการณ์ความสนใจ เหตุการณ์กำหนดได้จากเงื่อนไขที่คุณระบุ

ต่อไปนี้คือตัวอย่าง:

  • ตรวจหาการวัดพายุเฮอริเคนที่อยู่ในประเภท 4 โดยพิจารณาจากลักษณะความเร็วลม
  • เมื่อใช้การวัดจีพีเอสของการเคลื่อนที่ของเรือโดยฟิลด์แสดงความเร็วของเรือ ให้หาการวัดที่มากกว่าค่ามีนของการวัดห้าครั้งก่อนหน้านี้ 1.5 เท่า


ตำแหน่งจีโอโค้ดที่ได้จากตาราง

แปลงที่อยู่เป็นพิกัด ใช้เครื่องมือนี้บนตารางแบ่งปันไฟล์ข้อมูลขนาดใหญ่


ค้นหาตำแหน่งที่ใกล้เคียง

ตามเกณฑ์ที่คุณระบุหาสถานที่ที่คล้ายกันโดยการวัดความคล้ายคลึงกันของสถานที่ตั้งที่อยู่ในชั้นข้อมูลที่ใช้ในการค้นหาของคุณให้เป็นสถานที่หนึ่ง ใช้ในการอ้างอิง

ต่อไปนี้คือตัวอย่าง:

  • ค้นหาร้านค้าสิบร้าน ที่คล้ายกันมากที่สุด โดยการตรวจสอบจากจำนวนพนักงาน และการขายประจำปี
  • ค้นหา 100 เมืองคล้ายกันมากที่สุด โดยการตรวจสอบความสัมพันธ์ระหว่างการเติบโตของประชากรประจำปีและรายได้ภาษี


วิเคราะห์รูปแบบเชิงพื้นที่

เครื่องมือเหล่านี้มีหน้าที่ในการระบุตัวตน คำนวณปริมาณ และแสดงผลลัพธ์ข้อมูลของคุณในรูปแบบเชิงพื้นที่

  • คำนวณความหนาแน่น ตอบคำถามเกี่ยวกับจำนวนเหตุการณ์และการกระจายตัวของเหตุการณ์เหล่านั้นผ่านแผนที่
  • ค้นหาจุดฮอตสปอต ระบุจากข้อมูลรูปแบบเชิงพื้นที่ที่ได้จากการรวบกลุ่มทางสถิติที่สำคัญ
  • Find Point Clusters จะค้นหากลุ่มของฟีเจอร์จุด ภายในเสียงแวดล้อมโดยอิงอยู่กับการกระจายในเชิงพื้นที่

คำนวณความหนาแน่น

เครื่องมือคำนวณความหนาแน่นสร้างแผนที่ความหนาแน่นจากฟีเจอร์แบบจุด โดยการกระจายปริมาณที่รู้จักกันของปรากฏการณ์ (แสดงเป็นลักษณะของจุด) ในแผนที่ ผลที่ได้คือชั้นของพื้นที่ที่เป็นตัวแทนของความหนาแน่น

ต่อไปนี้คือตัวอย่าง:

  • การคำนวณความหนาแน่นของโรงพยาบาลที่อยู่ในพื้นที่หนึ่ง ๆ ชั้นข้อมูลผลลัพธ์จะแสดงเส้นทางที่สามารถเข้าถึงโรงพยาบาลได้ทั้งที่ง่ายและยาก และข้อมูลเหล่านี้จะช่วยในการตัดสินใจว่า ควรจะสร้างโรงพยาบาลแห่งใหม่ ณ ตำแหน่งใด
  • การระบุพื้นที่ที่มีความเสี่ยงต่อการเกิดไฟป่าสูง โดยพิจารณาจากตำแหน่งที่เคยมีไฟป่าเกิดขึ้น
  • การระบุตำแหน่งชุมชนที่ตั้งอยู่อย่างห่างไกลจากถนนสายหลัก เพื่อที่จะนำไปวางแผนการก่อสร้างถนนเส้นใหม่


Find Point Clusters

เครื่องมือ Find Point Clusters จะค้นหากลุ่มของฟีเจอร์จุดภายในเสียงแวดล้อมโดยอิงอยู่กับการกระจายในเชิงพื้นที่

ต่อไปนี้คือตัวอย่าง:

  • ค้นหากลุ่มของแมลง-ที่ระบาดในครัวเรือนเพื่อช่วยดำเนินความพยายามทำลายล้างเป้าหมาย
  • แจ้งและดำเนินการช่วยเหลือและอพยพตามจำเป็นโดยดูจากขนาดและตำแหน่งที่ตั้งของกลุ่มโดยใช้การทวีตตามตำแหน่งทางภูมิศาสตร์ติดตามอันตรายตามธรรมชาติหรือการโจมตีเพื่อก่อการร้าย


ค้นหาจุดเฝ้าระวัง

เครื่องมือค้นหาจุดสูงสุดจะตรวจสอบว่า ถ้ามีการรวมกลุ่มของค่าสถิติอย่างมีนัยยะสำคัญโดยใช้แบบรูปเชิงพื้นที่ของข้อมูลของคุณ

  • จุดของคุณ (เหตุการณ์อาชญากรรม, ต้นไม้, การเกิดอุบัติเหตุจราจร) มีการร่วมกลุ่มหรือไม่ คุณสามารถแน่ใจได้อย่างไร
  • คุณเชื่อการค้นพบอย่างแท้จริงของค่าสูงสุดอย่างมีนัยสำคัญทางสถิติ (สำหรับการใช้จ่าย, การเสียชีวิตของทารก ,คะแนนการทดสอบสูงอย่างต่อเนื่อง) หรือถ้าหากแผนที่ของคุณเล่าถึงการเปลี่ยนแปลงเรื่องราวได้ เพียงแค่คุณเปลี่ยนวิธีการใช้สัญลักษณ์
เครื่องมือค้นหาจุดสูงสุดจะช่วยคุณตอบคำถามเหล่านี้ได้อย่างมั่นใจ


วิเคราะห์พื้นที่ข้างเคียง

เครื่องมือนี้ช่วยตอบคำถามที่พบบ่อย เกี่ยวกับการวิเคราะห์เชิงพื้นที่ อะไรคือสิ่งที่อยู่ใกล้กับอะไร?

  • สร้างบัฟเฟอร์ สร้างพื้นที่ด้วยระยะที่กำหนดจากข้อมูล

สร้างพื้นที่ขยาย

บัฟเฟอร์เป็นพื้นที่ที่ครอบคลุมระยะทางที่กำหนดจากจุด เส้น หรือ รูปหลายเหลี่ยม

ต่อไปนี้คือตัวอย่าง:

  • การใช้ฟิเจอร์ของแม่น้ำเส้น บัฟเฟอร์แต่ละแม่น้ำออกมา 50 เท่า ความกว้างของแม่น้ำเพื่อกำหนดขอบเขตชายฝั่ง
  • พื้นที่ที่ได้รับเป็นตัวแทนของประเทศบัฟเฟอร์แต่ละประเทศ 200 ไมล์ทะเลเพื่อตรวจสอบอาณาเขตทะเล


จัดการข้อมูล

เครื่องมือเหล่านี้จะใช้สำหรับการจัดการแบบวันต่อวันของข้อมูลทางภูมิศาสตร์และตาราง

  • ข้อมูลภาคผนวก ผนวกฟีเจอร์ต่าง ๆ เข้ากับชั้นข้อมูลฟีเจอร์ที่โฮสต์อยู่ในปัจจุบัน
  • คำนวณฟิลด์ สร้างฟิลด์ใหม่หรือแก้ไขฟิลด์ที่มีอยู่ และสร้างชั้นข้อมูลใหม่ในเนื้อหาของคุณ
  • คัดลอกไปที่เก็บข้อมูล คัดลอกข้อมูลไปที่ ArcGIS Data Store ของคุณ และสร้างชั้นข้อมูลในเนื้อหาของคุณ
  • การซ้อนทับชั้นข้อมูล ผสมผสานชั้นข้อมูล 2 ชั้น หรือมากกว่านั้นให้เหลือเพียงชั้นข้อมูลเดียว

คัดลอกไป Datastore

สำเนาชั้นข้อมูลอินพุตหรือตารางต่อไปยัง ArcGIS Data Store และสร้างชั้นข้อมูลในเนื้อหาของคุณใน ArcGIS Enterprise

ต่อไปนี้คือตัวอย่าง:

  • คัดลอกคอลเลกชันของ .csv ไฟล์ข้อมูลขนาดใหญ่จะถูกแบ่งพื้นที่ในการจัดเก็บข้อมูล ในพื้นที่ชั่วคราวสำหรับการแสดงผล
  • คัดลอกฟีเจอร์ในขอบเขตแผนที่ปัจจุบันที่จัดเก็บไว้ในการจัดเก็บข้อมูลเชิงพื้นที่ชั่วคราวในการจัดเก็บข้อมูลเชิงสัมพันธ์


การคำนวณสนาม

คำนวณค่าสำหรับฟิลด์ใหม่หรือฟิลด์ที่มีอยู่ และสร้างชั้นข้อมูลในเนื้อหาของคุณใน ArcGIS Enterprise

ต่อไปนี้คือตัวอย่าง:

  • แก้ไขฟิลด์ที่มีอยู่ที่ชื่อ total เพื่อเป็นผลรวมของรายได้จากฟิลด์ total_2016, total_2017 และ total_2018
  • สร้างฟิลด์ใหม่เพื่อจัดระดับอันตรายโดยพิจารณาจากค่าต่าง ๆ ในฟิลด์ เช่น windspeed และ pollutant


ซ้อนทับชั้นข้อมูล

การซ้อนทับชั้นข้อมูล คือการผสมผสานชั้นข้อมูล 2 ชั้น หรือมากกว่านั้นให้เหลือเพียงชั้นข้อมูลเดียว การซ้อนทับถูกใช้เพื่อค้นหาคำตอบให้แก่คำถามพื้นฐานที่สุดทางภูมิศาสตร์ อะไรอยู่เหนืออะไร ต่อไปนี้คือตัวอย่าง:

  • พื้นที่ใดบ้างที่ถูกน้ำท่วมในช่วง 100 ปีที่ผ่านมา (การตั้งคำถามต้องมีการระบุขอบเขต ระยะเวลา ตำแหน่งที่ตั้งให้ชัดเจน)
  • ดินประเภทนี้ มีการใช้ที่ดินอย่างไรบ้าง
  • มีอะไรหลงเหลืออยู่ในหลุมของทหารที่ถูกทิ้งร้างบ้าง


ข้อมูลภาคผนวก (Append Data)

ผนวกฟีเจอร์เข้ากับชั้นข้อมูลที่ถูกโฮสต์ที่มีอยู่ลงในเนื้อหาของคุณใน ArcGIS Enterprise

ต่อไปนี้คือตัวอย่าง:

  • ปรับปรุงชั้นข้อมูลฟีเจอร์ 2018_Sales ของคุณทุกเดือนเมื่อรายงานของคุณพร้อมใช้งาน
  • ผนวกชุดข้อมูลการติดตามพายุเฮอริเคนจากปี 2016 และ 2017 ลงในชุดข้อมูลพายุเฮอริเคนหลัก
  • รวบรวมและผนวกชุดข้อมูลจากหลาย ๆ แหล่งที่มีสคีมาที่แตกต่างกันโดยใช้การแมปฟิลด์แบบกำหนดเอง
