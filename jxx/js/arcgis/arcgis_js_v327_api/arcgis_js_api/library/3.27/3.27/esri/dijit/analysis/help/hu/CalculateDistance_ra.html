Távolság kiszámítása

This tool calculates Euclidean distance, direction, and allocation from a single source or set of sources.

Az alkalmazásra példa lehet az alábbihoz hasonló kérdések megválaszolása:

  • Mekkora a távolság a legközelebbi városhoz?
  • Mekkora a távolság a legközelebbi úthoz?
  • Mekkora a távolság a legközelebbi vízforráshoz?

Az Aktuális térképkiterjedés használata bejelölésével az elemzés csak azokat a bemenő rétegen lévő vektoros elemeket foglalja magában, amelyek láthatók az aktuális térképkiterjedésen. Ha nincs bejelölve, akkor a bemenő réteg összes vektoros elemét elemzi, akkor is, ha azok kívül esnek az aktuálisan megjelenített térképkiterjedésen.

Válasszon raszteres vagy vektoros helyréteget a távolság kiszámításához

Azokat a kiindulópontokat meghatározó réteg, amelyektől a távolságot számítani kell. A réteg lehet képszolgáltatás vagy vektoros adatszolgáltatás.

Képszolgáltatás esetén a bementi típus lehet integer vagy lebegőpontos érték.

Vektoros adatszolgáltatás esetén a bemenet lehet pont, vonal vagy poligon.

Kiszámítandó maximális távolság

A kiszámítandó maximális távolság. Ha például értékként 10 000 métert ad meg, az euklideszi távolság a kiindulóhelyektől ekkora értékre (távolságra) lesz kiszámítva.

Ha nem ad meg értéket, a távolságot a meghatározott kiterjedésig számítja.

A rendelkezésre álló mértékegységek: kilométer, méter, mérföld és láb.

Kimeneti cellaméret

Enter the cell size and unit for the output rasters.

The units can be in Kilometers, Meters, Miles, or Feet.

Eredményül kapott távolságréteg neve

The name of the layer that will be created in My Content and added to the map. Az alapértelmezett név az eszköz nevétől és a bemenő réteg nevétől függ. Ha a réteg már létezik, akkor a program megkéri, hogy adjon meg egy másik nevet.

Az Eredmény mentési helye legördülő mezőben megadhatja a Saját tartalom mappájának nevét, ahová az eszköz menti az eredményt.

Eredményül kapott irányréteg neve

The name of the layer that will be created in My Content. You must provide a name that does not already exist in My Content.

Az Eredmény mentési helye legördülő mezőben megadhatja a Saját tartalom mappájának nevét, ahová az eszköz menti az eredményt.

Eredményül kapott allokációréteg neve

The name of the layer that will be created in My Content. You must provide a name that does not already exist in My Content.

Az Eredmény mentési helye legördülő mezőben megadhatja a Saját tartalom mappájának nevét, ahová az eszköz menti az eredményt.

Allokáció kiválasztása mező

Az egyes kiindulópontokat meghatározó értékeket tartalmazó mező a bemeneti kiindulópont-rétegen. Lehet a kiindulópontok adathalmazának integer- vagy szöveges mezője.

Az alapértelmezett érték Value. Ha vektoros adatszolgáltatás használata esetén nem érhető el Value mező, akkor az első elérhető integer- vagy szöveges mezőt használja. Ha nem létezik érvényes mező, az ObjectID mezőt (például OID vagy FID, a vektoros bemenet típusától függően) használja.
