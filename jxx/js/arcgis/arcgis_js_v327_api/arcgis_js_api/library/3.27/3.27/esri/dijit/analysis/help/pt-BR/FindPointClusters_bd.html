Localizar Agrupamentos de Pontos

A ferramenta Localizar Agrupamentos de Pontos encontra conjuntos de feições de ponto dentro do ruído circundante com base em sua distribuição espacial.

Uma organização não governamental está estudando uma doença específica transmitida por pragas. Ela tem um conjunto de dados de ponto representando as famílias em uma área de estudo, algumas das quais estão infestadas e outras não. Ao utilizar a ferramenta Localizar Agrupamentos de Pontos , um analista pode determinar grupos de famílias infestados para ajudar a identificar uma área para iniciar o tratamento e o extermínio de pragas.

Escolher a camada na qual os agrupamentos serão calculados

A camada de ponto na qual os agrupamentos serão encontrados. As camadas devem estar em uma referência espacial plana ou ter uma referência espacial de processamento definida para um sistema de coordenadas planas utilizando Ambientes de Análise.

Além de escolher uma camada do seu mapa, você pode escolher Procurar Camadas na parte inferior da lista suspensa para procurar por seu conteúdo para um conjunto de dados de compartilhamento do arquivo de grandes dados ou camada de feição.

Número mínimo de pontos a ser considerado como um agrupamento

O número mínimo de feições para propagar um agrupamento.

Limitar o intervalo de pesquisa para

A distância máxima para considerar feições para fazerem parte do mesmo agrupamento.

Nome da camada resultante

O nome da camada que será criada em Meu Conteúdo e adicionada ao mapa. O nome padrão é baseado no nome da ferramenta e o nome da camada de entrada. Se a camada já existe, a ferramenta falhará.

Ao utilizar a caixa suspensa Salvar resultado em, é possível especificar o nome de uma pasta em Meu Conteúdo onde o resultado será salvo.
