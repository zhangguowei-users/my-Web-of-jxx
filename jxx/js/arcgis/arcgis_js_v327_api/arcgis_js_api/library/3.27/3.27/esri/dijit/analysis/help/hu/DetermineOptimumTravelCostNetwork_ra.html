Optimális útiköltséghez tartozó hálózat meghatározása

This tool calculates the optimum cost network from a set of input regions.

Az eszköz alkalmazására lehet példa a legjobb hálózat megtalálása a megkülönböztetett gépjárművek számára.

Az Aktuális térképkiterjedés használata bejelölésével az elemzés csak azokat a bemenő rétegen lévő helyeket foglalja magában, amelyek láthatók az aktuális térképkiterjedésen. Ha nincs bejelölve, akkor a bemenő réteg összes helyét elemzi, akkor is, ha azok kívül esnek az aktuális térképkiterjedésen.

Raszteres vagy vektoros helyréteg kiválasztása az útvonalak meghatározásához a következők között:

A legkisebb költséggel járó hálózattal összekapcsolni kívánt bemeneti régiók.

A régiók meghatározásához használható képszolgáltatás vagy vektoros adatszolgáltatás.

Képszolgáltatás használata esetén a régiókat azonos értékű szomszédos cellák csoportjai határozzák meg. Mindegyik régiónak egyedi számozással kell rendelkeznie. Az egyetlen régió részét sem képező celláknak NincsAdat értékkel kell rendelkezniük. A rasztertípusnak integernek kell lennie, az értékek pedig lehetnek pozitívak vagy negatívak.

Vektoros adatszolgáltatás használata esetén a bemenet lehet pont, vonal vagy poligon.

Raszteres útiköltségréteg kiválasztása

A raster defining the impedance or cost to move planimetrically through each cell.

The value at each cell location represents the cost-per-unit distance for moving through the cell. Each cell location value is multiplied by the cell resolution while also compensating for diagonal movement to obtain the total cost of passing through the cell.

The values of the cost raster can be integer or floating point, but they cannot be negative or zero (you cannot have a negative or zero cost).

Eredményül kapott optimális útvonalak vonalláncrétegének neve

The name of the layer that will be created in My Content and added to the map. Az alapértelmezett név az eszköz nevétől és a bemenő réteg nevétől függ. Ha a réteg már létezik, akkor a program megkéri, hogy adjon meg egy másik nevet.

Az Eredmény mentési helye legördülő mezőben megadhatja a Saját tartalom mappájának nevét, ahová az eszköz menti az eredményt.
