Legg lag oppå

Legg lag oppå kombinerer to eller flere lag til ett lag. Et overlegg blir som om du ser ned gjennom en stabel med kart som ligger oppå hverandre, og oppretter et enkelt kart som inneholder all den informasjonen som finnes i hele kartstabelen. Overlegg er mye mer enn en sammenslåing av streker. Alle attributtene for geoobjektene som er en del av overlegget, videreføres til det endelige produktet. Overlegg brukes til å besvare et av de mest grunnleggende geografiske spørsmålene: «Hva ligger oppå hva?» Her følger noen eksempler:

  • Hvilke tomter ligger innenfor flomsonen for hundreårsflom? (Innenfor betyr det samme som oppå.)
  • Hvilken arealbruk er det oppå hvilken jordart?
  • Hvilke brønner befinner seg på nedlagte militærbaser?

Hvis det er merket av for Bruk gjeldende kartutstrekning, er det bare geoobjektene i inndata- og overleggslaget som er synlige innenfor gjeldende kartutstrekning, som legges over. Hvis det ikke er merket av for dette alternativet, legges alle geoobjektene i både inndatalaget og overleggslaget over, selv om de befinner seg utenfor gjeldende kartutstrekning.

Velg inndatalag

Punkt-, linje- eller områdelaget som det skal legges over.

I tillegg til å velge et lag fra kartet kan du velge Velg analyselag i levende atlas eller Velg analyselag som du finner nederst i rullegardinlisten. Dette åpner et galleri som inneholder en samling lag som vil være nyttige i mange analyser.

Velg overleggslag

Laget som skal legges over inndatalaget.

I tillegg til å velge et lag fra kartet kan du velge Bla gjennom lag, som du finner nederst i rullegardinlisten, til å bla deg frem til innholdet for et fildelingsdatasett for big data eller et geoobjektslag.

Velg overleggsmetode

Overleggsmetoden definerer hvordan inndatalaget og overleggslaget kombineres.

  • Intersect—The result will contain only those features or portions of features in the overlay layer that overlap with features in the input layer. Ved kryssende linje- eller områdegeoobjekter har du muligheten til å angi hvilken type geoobjekter du ønsker som resultat.

    Punkter med punkter

    Linjer med linjer

    Områder med områder

  • Erase—The result will contain features or portions of features in the overlay layer that do not overlap with the areas in the input layer. Overleggslaget kan inneholde område-, linje- eller punktgeoobjekter.

Navn på resultatlag

Navnet på laget som opprettes i Mitt innhold og legges til kartet. Det har samme geoobjektstype (områder, linjer eller punkter) som inndata- og overleggslagene. Standardnavnet er basert på overleggsmetoden og navnet på inndatalaget. Hvis laget finnes allerede, blir du bedt om å oppgi et annet navn.
