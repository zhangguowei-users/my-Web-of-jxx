Rasti taškų sankaupas

Įrankis Rasti taškų sankaupas randa taškų elementų sankaupas juos supančioje aplinkoje (triukšme) pagal jų pasiskirstymą erdvėje.

Nevalstybinė organizacija tyrinėja tam tikrą kenkėjų platinamą ligą. Ji turi taškų duomenų rinkinį iš tiriamos teritorijos su namų ūkiais, kai kurie iš jų užkrėsti, o kiti – ne. Naudodamas Rasti taškų sankaupas įrankį, analitikas gali nustatyti užkrėstų namų ūkių sankaupas ir nurodyti plotą, kur reikėtų pradėti naikinti kenkėjus.

Pasirinkite sluoksnį, kurio sankaupų bus ieškoma

Taškinis sluoksnis, kuriame bus ieškoma sankaupų. Sluoksniai turi turėti projekcinę koordinačių sistemą arba projekcinė koordinačių sistema turi būti nustatyta naudojant Analizės aplinką.

Galite ne tik nurodyti sluoksnį iš žemėlapio, bet ir apačioje pasirinkti Naršyti sluoksnius, kad galėtumėte turinyje naršyti didelių duomenų rinkinio arba elemento sluoksnio.

Rezultatų sluoksnio pavadinimas

Sluoksnio, kuris bus sukurtas puslapyje Turinys ir pridėtas į žemėlapį, pavadinimas. Numatytasis pavadinimas sukuriamas pagal įrankio ir įvesties sluoksnio pavadinimus. Jei toks sluoksnis jau yra, įrankis neveiks.

Iššokančiame meniu Įrašyti rezultatą į galite nurodyti Turinyje esančio aplanko, kur bus įrašyti rezultatai, pavadinimą.
