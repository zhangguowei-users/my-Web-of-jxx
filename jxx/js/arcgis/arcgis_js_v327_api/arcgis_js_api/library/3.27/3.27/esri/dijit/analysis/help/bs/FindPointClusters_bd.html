Pronađi klastere točaka

Alat Pronađi klastere točaka pronalazi klastere točkastih geoobjekata unutar okolnih podataka na temelju njihove prostorne distribucije.

Nevladina organizacija proučava određenu bolest koju prenose nametnici. Ima točkasti skup podataka koji predstavlja kućanstva u radnom području od kojih su neka zaražena, a neka nisu. Upotrebom alata Pronađi klastere točaka analitičar može odrediti klastere zaraženih kućanstava kako bi se pomoglo odrediti područje za početak tretiranja i istrebljivanja nametnika.

Izaberite sloj za koji će se pronaći klasteri

Točkasti sloj u kojem se nalaze klasteri. Slojevi moraju biti u predviđenoj prostornoj referenci ili imati prostornu referencu koja se obrađuje postavljenu u predviđenom koordinatnom sustavu pomoću Okruženja za analizu.

Osim odabira sloja s vaše karte, možete odabrati Pretraži slojeve na dnu padajućeg popisa za pretraživanje vaših sadržaja za skup podataka za dijeljenje datoteka velikih podataka ili sloj geoobjekta.
