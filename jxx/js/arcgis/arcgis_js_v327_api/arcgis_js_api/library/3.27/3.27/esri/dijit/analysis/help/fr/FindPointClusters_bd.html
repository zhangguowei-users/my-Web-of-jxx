Rechercher des agrégats de points

L’outil Rechercher des agrégats de points détecte les agrégats d’entités ponctuelles dans le bruit environnant en se basant sur leur distribution spatiale.

Une organisation non gouvernementale étudie une maladie parasitaire particulière. Vous disposez d’un jeu de données ponctuelles représentant les ménages dans votre zone d’étude, certains d’entre eux étant touchés, et d’autres non. En utilisant l’outil Rechercher des agrégats de points, un analyste peut déterminer les agrégats de ménages infestés pour vous aider à localiser la zone où commencer le traitement et exterminer les parasites.

Choisir la couche pour laquelle les agrégats seront identifiés

La couche de points contenant les agrégats. Les couches doivent se trouver dans une référence spatiale projetée ou disposer d’une référence spatiale projetée définie sur un système de coordonnées projetées à l’aide de l’option Analysis Environments (Environnements d’analyse).

En plus de choisir une couche de votre carte, vous pouvez sélectionner Parcourir les couches au bas de la liste déroulante pour parcourir votre contenu et rechercher une couche d'entités ou un jeu de données de partage de fichiers Big Data.

Nombre minimal de points à considérer comme agrégat

Le nombre minimum d’entités pour constituer un agrégat.

Limiter la plage de recherche à

La distance maximale pour appartenir à un même agrégat.

Nom de la couche de résultat

Le nom de la couche qui sera créée dans la page My Content (Mon contenu) et ajoutée à la carte. Le nom par défaut repose sur le nom de l'outil et sur le nom de la couche en entrée. Si la couche existe déjà, l'outil échoue.

La zone de liste déroulante Save result in (Enregistrer le résultat dans) vous permet de spécifier le nom d’un dossier dans My content (Mon contenu) où le résultat sera enregistré.
