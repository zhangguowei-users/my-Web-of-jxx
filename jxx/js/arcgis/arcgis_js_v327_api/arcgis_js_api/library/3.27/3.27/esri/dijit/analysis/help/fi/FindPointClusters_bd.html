Etsi pisteklustereita

Etsi pisteklustereita -työkalu etsii pistekohdeklustereita ympäröivästä kohinasta sijainnin jakauman perusteella.

Eräs kansalaisjärjestö tutkii tiettyä tuhoeläinten aiheuttamaa sairautta. Siinä on pisteaineistoryhmä, joka edustaa tutkittavan alueen kotitalouksia. Osassa näistä on tuhoeläimiä. Etsi pisteklustereita -työkalun avulla analysoija voi määrittää tuhoeläinten valtaaman kotitalousklusterin kyseisen alueen paikantamiseksi, jotta tuhoeläinten torjunta voidaan aloittaa.

Valitse karttataso, jolle klusterit etsitään

Pistekarttataso, missä klusterit ovat. Karttatasojen on oltava projisoidussa koordinaattijärjestelmässä, tai käsittelevä koordinaatisto on määritettävä projisoituun koordinaattijärjestelmään käyttämällä analyysiympäristöjä.

Sen lisäksi, että voit valita karttatason kartasta, voit valita Selaa karttatasoja -vaihtoehdon avattavan luettelon alareunasta ja selata Big Data -tiedostoresurssin aineistoryhmän tai karttatason sisältöön.

Tulostason nimi

Sen karttatason nimi, joka luodaan Oma sisältö-kohtaan ja lisätään karttaan. Oletusnimi perustuu työkalun nimeen ja lähtökarttatason nimeen. Jos karttataso on jo olemassa, työkalua ei voi suorittaa.

Avattavan Tallenna tulos kohteeseen -valikon avulla voit määrittää sen Oma sisältö -kansion nimen, johon tulos tallennetaan.
